# Collection 10: Meta-Learning and Adaptability

This collection focuses on algorithms that "Learn how to Learn," handle multiple tasks, or evolve their own structures and schedules.

## 📈 Evolutionary Path
`MAML` ⮕ `Reptile` ⮕ `Meta-RL (RL^2)` ⮕ `AutoRL`

## 📂 Algorithms in this Collection
1. **Meta-RL (RL^2)**: Framing meta-learning as a reinforcement learning problem.
2. **MAML**: Model-Agnostic Meta-Learning for fast adaptation to new tasks.
3. **AutoRL**: Using evolutionary strategies to optimize the RL process itself.
4. **Policy Distillation**: Compressing knowledge from multiple experts into a single student.
5. **POET**: Open-ended evolution of environments and agents simultaneously.
